# Answer

## Answer Question No 1

Simple Query :
```
  SELECT
    a.ID,
    a.UserName,
    b.UserName AS ParentUserName
  FROM
    `USER` a
  LEFT JOIN `USER` b ON
    a.Parent = b.ID;
```
